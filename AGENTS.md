# Agent Instructions (Repo Defaults)

## Visual Artifacts (Diagrams, Wireframes)

When creating or updating **visual artifacts** (Mermaid, D2, ASCII wireframes) anywhere in this repo:

- Follow `docs/z.guidelines/visual-artifacts-rules.md` (canonical execution rules).
- Also apply the cross-cutting design constraints in `docs/z.guidelines/visual-guidelines-abstract.md` by default (Germany + i18n-first + mobile-first + white-label + healthcare sensibility).
- For concrete UI token choices (when relevant), use `docs/z.guidelines/visual-guideline.md`.
