Key Product Insight

Most teams over-invest in high-frequency, low-urgency features (polish) and under-invest in medium-frequency, high-urgency features (recovery, validation, edge cases). In healthcare, that inversion is fatal.

Decision Heuristic
  1. Fix the one with higher urgency, even if frequency is low.
  2. Ship the one with higher frequency only if urgency is equal.

Partner Integration

Co-branded experience where partners are visible but MedAlpha owns the user journey.

CUR

Curaay

In-person appointment booking. API for availability and booking confirmation.

TLC

Teleclinic

Telemedicine video consultations. Embedded video, questionnaire, and summary.

GEM

Cardlink / gematik

E-prescription retrieval via NFC. Scans eGK to extract prescription tokens.

APO

Apo Group

Online pharmacy and prescription delivery. Order placement and tracking.

MAP

Google Maps

Pharmacy search and directions. Location services for offline redemption.
