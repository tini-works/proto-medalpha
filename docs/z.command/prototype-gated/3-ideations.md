# Step: IDEATIONS

**Purpose:** Generate 4-6 distinct solution approaches with cross-domain inspiration

## Input

- Selected hypothesis from Step 1 / Gate 1
- IA and User Flow context (from Gate 1 or existing docs)
- Optional: Scope and UX interaction constraints

## Process

1. **Review context** - hypothesis, IA, user flow
2. **Generate 4-6 approaches** - each fundamentally different
3. **For each approach:**
   - Core concept (2-3 sentences)
   - Cross-domain inspiration (optional, max 3 total)
   - How it works (bullet points)
   - Trade-offs table (Pros vs Cons)
   - Best suited for (user type)
4. **Build comparison matrix**
5. **Recommend** - AI suggests best approach with rationale
